Version Changelog

Track all updates, new features, and improvements across Waypoint Planner versions

v1.2.0 Current
December 2024

New Features

  • Photoshop-Style Interface - Complete UI redesign with draggable, floating panels that can be moved and resized anywhere on screen
  • Advanced Photogrammetry Tools - Comprehensive photogrammetry calculator including GSD (Ground Sample Distance), coverage area calculator, image count estimator, processing time calculator, and camera settings recommendations
  • Offline-First Architecture - Full offline functionality with local-first data storage using IndexedDB, automatic background sync when online, and Progressive Web App (PWA) capabilities
  • Keyboard Shortcuts - Global keyboard shortcuts for common actions (Ctrl+S to save, etc.)
  • Dark/Light Theme Toggle - System-wide theme switching with enhanced dark mode
  • Customizable UI Layouts - Four different layout options (Default, Compact, Wide, Minimal) accessible from Settings panel
  • Waypoint Search/Filter - Search and filter waypoints by various criteria
  • Measurement Tools - Distance and area measurement tools directly on the map
  • Waypoint Clustering - Visual clustering of waypoints for large missions
  • Print-Friendly Flight Plan View - Generate printable flight plan reports
  • Drag-and-Drop File Import - Import flight plans by dragging files onto the application
  • Export Presets - Save and reuse custom export configurations
  • Floating Waypoint Editor - Non-intrusive waypoint editor that doesn't hide other information

Improvements

  • Enhanced dark theme with deeper, more professional color palette
  • Improved panel management with position persistence
  • Better organization of tools and settings
  • Optimized performance for large flight plans
  • Enhanced offline capabilities and data persistence
v1.1.0 Previous
July 2024

New Features

  • Welcome Page & Project Management - New welcome screen with project browser, recent projects list, and quick project creation interface
  • Auto-Save System - Automatic saving of flight plans to prevent data loss during work
  • Battery Usage Calculator - Estimate battery consumption and maximum safe flight distance based on drone model and flight parameters
  • Enhanced Export Options - Multiple export formats including CSV for spreadsheet analysis, comprehensive PDF flight reports, DJI FlightHub JSON format, and Litchi CSV format
  • Flight Statistics Dashboard - View detailed flight statistics including total distance, estimated flight time, battery usage, and waypoint count
  • Terrain Elevation Integration - Apply terrain elevation data to waypoints for accurate altitude planning
  • Waypoint Templates - Pre-configured waypoint patterns for common mission types
  • Toast Notification System - User-friendly notifications for actions, errors, and system messages
  • Professional Documentation - Complete user guide and professional website documentation

Improvements

  • Enhanced UI/UX with improved styling and user feedback throughout the application
  • Better error handling and user notifications for improved reliability
  • Improved waypoint management interface with better organization
  • Enhanced settings panel with better categorization and controls
  • Optimized build configuration for better performance
v1.0.0 Initial Release
2024

Core Features

  • Waypoint Creation - Create waypoints by clicking on the map or manually entering coordinates
  • DJI Drone Support - Comprehensive support for multiple DJI models including Mini 5 Pro, Mavic 4 Pro, Mini 4 Pro, Air 3/3S, and Mavic 3 series
  • Interactive Map - Visualize and edit flight plans on an interactive map with multiple map styles (Dark, Satellite, Terrain, Light, OpenStreetMap)
  • Flight Plan Management - Save and load flight plans as JSON files
  • Drawing Tools - Create waypoints from polygons, rectangles, and circular POI areas
  • Mission Splitting - Split large missions into smaller, manageable parts for battery management

Premium Features

  • Dynamic Altitude Adjustment - Automatically adjust altitude based on terrain elevation
  • Waypoint Actions - Add actions at waypoints (take photo, start/stop recording, rotate gimbal, hover)
  • Reverse Points - Reverse the order of waypoints in the flight path
  • Line Orientation - Rotate flight paths by specified angle (0-360°)
  • Straightened Flight Paths - Optimize paths by removing unnecessary waypoints
  • KMZ Import/Export - Import and export flight plans in KMZ format for DJI apps (DJI Fly, DJI Pilot)
  • KML/WGS84 Import - Import polygon data from KML files and coordinate files (WGS84, TXT)
  • Image Overlap Control - Configure forward and side overlap percentages for photogrammetry (0-100%)
  • Flight Presets - Quick configuration presets for 3D Modeling, Mapping, Scanning, and Inspection missions
